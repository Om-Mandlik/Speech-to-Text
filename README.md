# Speech to Text

A simple JavaScript-based voice recognition web application that listens to user speech and performs actions based on spoken commands. This project uses the **Web Speech API** (`SpeechRecognition`) to convert speech into text and execute predefined commands.

## 🚀 Features

- Speech-to-text conversion using browser speech recognition.
- Displays recognized speech on the webpage.


## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Web Speech API

## 📜 How It Works

1. User clicks the microphone button.
2. Speech recognition starts listening.
3. Spoken words are converted into text.
4. The recognized text is displayed on the screen.
5. If the command matches a predefined action, the corresponding task is executed.
